Por favor, coloque aqui a imagem do logo com o nome `logo.png` para que o app carregue automaticamente a partir do caminho `/logo.png`.

Recomendações:
- Nome do arquivo: logo.png
- Tamanho sugerido: 128x128 ou 64x64 (PNG com fundo transparente funciona melhor)
- Caminho final: `frontend/public/logo.png`

Depois de adicionar a imagem reinicie o servidor de desenvolvimento (na pasta frontend):

```powershell
npm run dev
```

Se preferir usar outro nome, ajuste o caminho em `src/components/sidebar.tsx`.
